# chats

一款轻量级的基于 SpringBoot + WebSocket 的在线聊天室项目，fork了 [Cool-CoCoder](https://github.com/Cool-CoCoder)的项目,
在[Cool-CoCoder](https://github.com/Cool-CoCoder)的聊天室基础上，将mysql 去掉 使用 sqlite 无需本地部署mysql，更便于维护和使用。

## 功能说明

1. 具备 登录/注册 用户的功能

2. 可以发送文字

3. 可以发送动图

4. 可以清屏
